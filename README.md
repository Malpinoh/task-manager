Task Manager App

A vanilla JavaScript task manager that allows users to create, manage, and prioritize tasks with persistence using localStorage. This project demonstrates DOM manipulation, event handling, state management, and interactive UI design—all essential skills for a JavaScript developer role.

Features
Add tasks with a title, description, and priority level (Low, Medium, High)
Mark tasks as completed or undo completion
Delete tasks individually
Filter tasks by status: All, Pending, Completed
Drag-and-drop reordering of tasks
Color-coded priorities for quick visual reference

Persistent storage: Tasks are saved in localStorage
Technologies Used
HTML5
CSS3 (responsive design)
Vanilla JavaScript (DOM manipulation, event handling, localStorage)
Usage

Clone the repository:
git clone https://github.com/YOUR_USERNAME/task-manager.git
Open index.html in a web browser.
Add tasks using the input fields and select a priority.
Drag tasks to reorder them or use buttons to complete/delete.
Filter tasks using the buttons: All, Pending, Completed.
